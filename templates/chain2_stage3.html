Need Help?
Hint 1: HTML forms can submit to any URL. Create a form with action="/change_password_vuln"
Hint 2: Use method="POST" and include a hidden input for the new password.
Hint 3: Add JavaScript to auto-submit: <body onload="document.forms[0].submit()">
