The Scenario

"You've found a server with a web-based diagnostic tool. It looks harmless - just a ping utility. But the developers made a fatal mistake: they didn't sanitize user input before passing it to the shell. This gives you a foothold. From here, explore the file system, map out the server, and ultimately exfiltrate sensitive data. This is a classic penetration testing scenario..."
