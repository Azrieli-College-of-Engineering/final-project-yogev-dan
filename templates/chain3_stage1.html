Need Help?
Hint 1: The shell executes: ping [your input]. What if you add more commands after ping?
Hint 2: On Windows, & chains commands. On Linux, ; or && work.
Hint 3: Try: 127.0.0.1 & whoami to run ping and then see who you're running as.
Command Injection Flow
Input
Shell
ping
whoami
Command: ping 127.0.0.1 & whoami
Shell sees: Run ping, THEN run whoami
Result: Attacker executes arbitrary commands!
Answer:
127.0.0.1 & whoami This pings localhost, then runs whoami to show the current user!
